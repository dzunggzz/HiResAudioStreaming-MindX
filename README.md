# *Sản Phẩm Cuối Khoá Web Development Advanced tại trung tâm MINDX*

HRAS là dự án nhằm mang lại trải nghiệm âm nhạc tuyệt vời tại gia. HRAS sử dụng API HIFI của [sachinsenal0x64](https://github.com/sachinsenal0x64/hifi) và CSS cơ bản cho phần UI.

Chất lượng âm thanh: HI-RES, HI-RES-LOSSLESS, LOSSLESS, HIGH, LOW. Tối đa đến 24-bit, 192kHz.
